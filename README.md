# GrappleGraph 🥋

A minimalist, dark-mode visual tool for mapping BJJ techniques, sequences, and positional hierarchies to master the why behind the move.

## 🚀 The Vision
GrappleGraph is designed to help practitioners visualize the "connective tissue" of BJJ. Instead of a linear list of moves, this tool uses a conceptual mind map to show how positions (like Guard or Side Control) flow into specific submissions or transitions.

## ✨ Features
* **Visual Mapping:** See the "why" behind transitions, not just the "how."
* **Positional Hierarchies:** Clear representation of dominant vs. neutral positions.
* **Minimalist Design:** Dark-mode focused to keep the focus on the data.

## 🛠️ Built With
* JavaScript
* AI-Assisted Development (Base44)

## 📁 How to Use
1. Clone this repository.
2. Open `index.html` in your browser.
3. Start mapping your rolls!
